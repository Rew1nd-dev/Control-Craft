# CompactFlapPeripheral Methods

This document describes the Lua methods available in the `CompactFlapPeripheral` peripheral.

### `getAngle`
- **参数**：
  - 无
- **返回值**：`double` - 角度，单位为度。
- **描述**：获取当前的角度值。


### `setAngle(angle)`
- **参数**：
  - `angle` (`double`): - 角度，单位为度。
- **返回值**：`void` - 
- **描述**：设置偏转角度值。
